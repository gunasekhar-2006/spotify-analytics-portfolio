# Spotify Songs Analytics Portfolio Project

## Overview
This project was completed as part of a Data Analytics Internship program.

The project focuses on analyzing Spotify songs data using:
- Python
- SQL
- Power BI
- Statistical Analysis
- Business Intelligence techniques

---

# Project Objectives

- Analyze Spotify music trends
- Study listener engagement patterns
- Explore audio feature relationships
- Build interactive dashboards
- Generate business insights

---

# Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQL
- Power BI
- SciPy

---

# Project Tasks

## Task 1 — Data Wrangling
- Data cleaning
- Missing value handling
- Duplicate removal
- Feature engineering

## Task 2 — Exploratory Data Analysis
- Genre analysis
- Popularity analysis
- Correlation analysis
- SQL business queries

## Task 3 — Interactive Dashboard
- KPI cards
- Scatter plots
- Genre charts
- Artist analysis
- Interactive filters

## Task 4 — Storytelling & Statistical Testing
- Hypothesis testing
- T-Test analysis
- Business recommendations
- Professional presentation development

---

# Key Insights

- Danceability significantly affects song popularity.
- Pop-related genres dominate Spotify engagement trends.
- Audio features strongly influence listener behavior.
- Interactive dashboards improve business intelligence analysis.
