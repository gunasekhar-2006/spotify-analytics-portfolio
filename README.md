# Spotify Analytics Portfolio

## Overview
This repository contains the complete Spotify Songs Analytics portfolio project developed during the Data Analytics Internship at ApexPlanet Software Pvt Ltd.

## Project Modules

### Task 1 — Data Wrangling
- Data cleaning
- Preprocessing
- Feature engineering

### Task 2 — EDA & SQL
- Exploratory Data Analysis
- SQL business intelligence queries
- Insights generation

### Task 3 — Power BI Dashboard
- KPI cards
- Interactive filters
- Genre and artist analysis

### Task 4 — Storytelling & Statistics
- Hypothesis testing
- Statistical validation
- Business presentation

## Technologies Used
- Python
- SQL
- Power BI
- Pandas
- Matplotlib
- Seaborn
- SciPy
