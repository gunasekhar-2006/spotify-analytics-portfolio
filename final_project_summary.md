# Final Project Summary

## Project Name
Spotify Songs Analytics

## Internship Organization
ApexPlanet Software Pvt Ltd

## Project Description
This project focuses on analyzing Spotify songs data using data analytics, business intelligence, and statistical techniques.

The project included:
- Data preprocessing
- Exploratory Data Analysis
- SQL querying
- Interactive dashboard development
- Statistical hypothesis testing
- Business storytelling

## Final Outcome
A complete end-to-end analytics solution was developed using Python, SQL, and Power BI.

The project successfully generated meaningful insights regarding:
- song popularity
- genre trends
- danceability impact
- artist performance
- listener engagement

## Conclusion
The project improved practical skills in analytics, visualization, dashboarding, and storytelling while solving a real-world business intelligence problem.
